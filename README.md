# Password Strength Analyzer 
## Overview
This project is a Python-based tool that analyzes the strength of a password based on length, character variety, and common security rules. It helps users understand how secure their passwords are against brute-force and dictionary attacks.
## Features
Checks password length
Detects uppercase, lowercase, numbers, and symbols
Estimates password strength. Weak,Medium
Strong
 Gives improvement suggestions
## Security Concepts
Password entropy
  Brute-force attack resistance
  Secure password creation principles
